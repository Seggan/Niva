
### Strings literals

### Bools

```
false and false // false
false or true // true
false not // true
```

### Ints & Floats

```C
1 + 1 // 2
5 / 2 // 2
5.0 / 2 // 2.5
```

### Variable declaration

```C
x = 42
y = x
x = 1123 // error X already exist
```

### Expression blocks
```Smalltalk
y = [ x::int | ^ x + 5 ]
(y value: 5) printnln // value - run block of code with x = 5, result is 10
```

### List
```C
{1, 2, 3}
```

### Tuples
```Smalltalk
(age: 0, name: "niva") // tuple[age: int, name: string]
//or
(0, "niva") // (int, string)
```
